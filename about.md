# 关于 Ay 开放平台

## 平台简介

Ay 开放平台是一个由个人开发者 **韵典（AeYunDian）** 打造的 API 服务集合。旨在为开发者、学习者、小型项目提供**简单、可靠、无需认证**的 API 接口。所有服务均免费开放，开箱即用。

平台目前提供 IP 归属地查询、QQ 好友一键添加、短链解析、文本暂存（AyPass）等实用功能，未来还将持续增加更多服务。

## 技术架构

- **运行环境**：Cloudflare Workers（全球边缘网络）
- **后端语言**：JavaScript (ES2022)
- **数据库**：Cloudflare D1 (SQLite)
- **缓存/存储**：Cloudflare KV
- **前端文档**：VitePress
- **域名**：undz.cn / io.hb.cn

所有 API 均部署在 Cloudflare 的边缘节点上，享受低延迟和高可用性。
