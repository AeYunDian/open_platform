# 添加 QQ 好友（一键拉起）

生成一个链接，移动端点击后直接拉起 QQ 客户端并跳转到添加好友页面。

## 接口信息

- **请求方法**：`GET`
- **请求路径**：`/addqq`

## 请求参数

| 参数 | 类型 | 必填 | 描述 |
|------|------|------|------|
| uid | string | 是 | 目标 QQ 号 |

## 响应说明

接口返回一个 HTML 页面，包含自动跳转逻辑。如果浏览器支持，会立即拉起 QQ；否则显示手动跳转提示。

## 调用示例

```
https://api.undz.cn/addqq?uid=123456789
```

## 注意事项

- 最新版 QQ 可能要求先打开主程序才能唤起添加页面。
